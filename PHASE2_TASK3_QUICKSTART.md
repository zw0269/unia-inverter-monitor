# Phase 2 Task 3 - 快速启动指南

## 🚀 快速开始

### 1. 访问电表监控

从首页有两种方式进入：

**方式1：直接点击首页指标**
```
首页 -> 点击"实时用电 kW"指标 -> 跳转到电表监控页面
```

**方式2：通过导航**
```
首页 -> 设备管理 -> 电表监控
```

### 2. 查看实时用电

电表监控页面显示：
- 📊 **实时用电功率**：当前瞬时功率（大字显示）
- ⚡ **今日用电量**：今天累计用电量
- 🔌 **电气参数**：电压、电流、功率因数、频率
- 💰 **今日电费**：今日累计电费

### 3. 查看能量平衡

在电表监控页面可以看到：
- 🔄 **能量流动图**：发电、用电、自用、上网、取电的关系
- 📈 **自发自用率**：中心圆圈显示百分比
- 📊 **能量数据**：各项能量数值

### 4. 用电分析

点击"用电分析"按钮进入：
```
电表监控 -> 点击"📊 用电分析"
```

查看内容：
- 📊 **用电统计**：总量、峰谷平分时用电
- 💰 **电费统计**：总费用、分时电费
- 📈 **电费占比**：饼图显示峰谷平占比
- 📉 **用电趋势**：堆叠柱状图显示历史趋势
- 🔋 **自发自用分析**：自用率、节省电费
- 💡 **用电建议**：智能优化建议

### 5. 查看电费账单

点击"电费账单"按钮进入：
```
电表监控 -> 点击"📃 电费账单"
```

查看内容：
- 📋 **账单摘要**：周期、用电量、总电费
- 💵 **费用明细**：峰谷平分项电费
- 🔋 **收益抵扣**：自用节省、上网收益
- 📜 **历史账单**：按月查看历史记录

## 🎯 使用场景

### 场景1：查看今日用电情况
```
1. 打开 App
2. 首页查看"今日用电 kWh"和"实时用电 kW"
3. 点击"实时用电"进入电表监控
4. 查看详细的用电数据和电费
```

### 场景2：分析用电模式
```
1. 电表监控 -> 点击"用电分析"
2. 选择时间周期（今日/本周/本月）
3. 查看峰谷平用电分布
4. 查看用电建议
5. 根据建议调整用电时间
```

### 场景3：查询电费账单
```
1. 电表监控 -> 点击"电费账单"
2. 选择查询周期（本月/近3月/近6月）
3. 查看账单摘要和费用明细
4. 查看发电收益抵扣
5. 导出账单（如需要）
```

### 场景4：优化用电成本
```
1. 查看用电分析的"用电建议"
2. 识别峰时高耗电设备
3. 将可调整的负载转移到谷时
4. 在光伏发电时段增加用电（提高自用率）
5. 定期查看电费变化
```

## 💡 使用技巧

### 技巧1：提高自发自用率
- 在光伏发电时段（6:00-18:00）增加用电
- 使用储能设备存储多余电量
- 将大功率设备使用时间与发电时段对齐

### 技巧2：降低电费成本
- 将非必要负载转移到谷时（00:00-06:00）
- 避免在峰时（08:00-11:00, 18:00-22:00）使用高功率设备
- 利用自发电量，减少电网取电

### 技巧3：监控用电异常
- 定期查看实时功率，识别异常波动
- 对比历史数据，发现用电异常
- 查看用电建议，及时调整

### 技巧4：电费预算管理
- 定期查看账单，了解用电成本
- 设置用电目标，控制月度电费
- 对比不同月份，分析用电趋势

## 📊 数据说明

### 电表数据来源
当前使用模拟数据，包括：
- 实时用电数据（每5秒更新）
- 历史用电数据（最近30天）
- 电费账单数据（最近6个月）

### 电价配置
默认电价（可在"电价配置"中修改）：
- 峰时电价：¥0.8/kWh（08:00-11:00, 18:00-22:00）
- 谷时电价：¥0.3/kWh（00:00-06:00）
- 平时电价：¥0.5/kWh（其他时段）
- 上网电价：¥0.35/kWh
- 光伏补贴：¥0.42/kWh

### 计算公式

**电费计算：**
```
峰时电费 = 峰时用电量 × 峰时电价
谷时电费 = 谷时用电量 × 谷时电价
平时电费 = 平时用电量 × 平时电价
总电费 = 峰时电费 + 谷时电费 + 平时电费
```

**能量平衡：**
```
自用量 = min(发电量, 用电量)
上网量 = max(0, 发电量 - 用电量)
取电量 = max(0, 用电量 - 发电量)
自发自用率 = (自用量 / 发电量) × 100%
```

**收益计算：**
```
自用收益 = 自用量 × 平均电价
上网收益 = 上网量 × 上网电价
补贴收益 = 总发电量 × 补贴电价
总收益 = 自用收益 + 上网收益 + 补贴收益
```

**实际电费：**
```
实际应缴 = 总电费 - 自用节省 - 上网收益
```

## 🔧 常见问题

### Q1: 为什么电表数据与发电数据不一致？
A: 电表记录的是用电量，发电数据记录的是光伏发电量，两者是不同的数据。能量平衡图展示了它们之间的关系。

### Q2: 自发自用率是如何计算的？
A: 自发自用率 = (自用电量 / 总发电量) × 100%。例如发电100kWh，用电70kWh，则自用70kWh，自用率为70%。

### Q3: 为什么有时显示"取电"，有时显示"上网"？
A: 当用电量大于发电量时，需要从电网取电；当发电量大于用电量时，多余电量上网。

### Q4: 如何修改电价配置？
A: 进入"收益统计" -> "电价配置"，可以修改峰谷平电价和时段。

### Q5: 账单数据可以导出吗？
A: 点击"导出账单"按钮即可导出（功能开发中）。
